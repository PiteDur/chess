ChessGameViewer - usage guide

This file explains how to use the ChessGameViewer package and the run_chess_game script.

1. Installation
   - Ensure Python 3.10+ is installed.
   - Install pygame and numpy if needed:
     pip install pygame numpy

2. Run the game with run_chess_game
   - Open a terminal in the project folder.
   - Execute:
     python run_chess_game.py
   - The game starts from the initial chess position.

3. Usage examples
   - Standard launcher:
     from chess import board
     from chess_viewer.chess_game_viewer import ChessGameViewer

     game_board = board()
     viewer = ChessGameViewer(game_board)
     viewer.start_game()

   - Custom starting position:
     from chess import board
     from chess_viewer.chess_game_viewer import ChessGameViewer

     game_board = board()
     positions = {
         'P': ['e4', 'f2'],
         'p': ['e5', 'f7'],
         'K': ['e1'],
         'k': ['e8']
     }
     custom_board = game_board.create_board(positions=(positions, 'w'))
     viewer = ChessGameViewer(game_board)
     viewer.current_board = custom_board
     viewer.run()

4. Controls
   - Click a piece to select it.
   - Click a green destination to move the piece.
   - The sidebar shows scores and captured material.

5. End of game
   - The viewer displays an overlay message when the game ends.
   - Close the window or press Ctrl+C to exit.
