# Chess: programing an application from end to end

This Repository [STILL UNDER CONSTRUCTION] presents a naive way to built a chess game from scratch in python. The goal is to get familiar with classes in python and to have an introduction in building visual application. 

The game is meant to be used in developping a small algorithm capable of playing. 

# Getting Started

## Access to the code

You may use an environment on your own computer (see `INSTALL.txt` for more information about conda environment), after having cloned this repository with :

``````
git clone https://github.com/PiteDur/chess
``````
